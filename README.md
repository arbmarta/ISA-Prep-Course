# Tree Fertilization Practice Website

A static, classroom-ready fertilizer math worksheet that can be hosted for free with GitHub Pages.

## Files

- `index.html` — page structure and worksheet content
- `styles.css` — visual design and mobile layout
- `script.js` — answer checking, scoring, hints, and reset controls

## Publish with GitHub Pages

1. Create a new GitHub repository, for example `tree-fertilization`.
2. Upload all four files from this folder to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and the `/root` folder, then save.
6. GitHub will display the public website address after deployment.

## Customize the questions

Open `script.js` and edit the `rows` or `bagProblems` arrays near the top of the file.
